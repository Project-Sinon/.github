# Project Koda

Project Koda is so sigma bro

## Projects

**depot** - macOS CLI made in *Swift* to install and manage the Procursus apt/dpkg bootstrap under /opt/procursus \
**Hecate** - Bare metal multi OS switching without virtualization, based on NetBSD - private development for now \
**KodaDarwin** - Run Darwin/XNU on a VM - private development for now \
**FreeBSD-T2** - Run FreeBSD on T2 Intel Macs - private development for now
